Help

Join is a Kanban Board. It visualises the project flow in agile project management.

This app was built as a training project at Developer Akademie. There is no security layer implemented. It is not supposed to be used with real life data.

Do not enter personal data.

Add a task

Create a new task in the add-task section.
Fill the form and choose by whom the task should be processed.
Decide whether the task should end up in the board for direct processing or in the backlog first.

The board

On the board each task passes through 4 sections.
The tasks can simply be moved by drag and drop.
After a task has been completed, it can either be archived or deleted.

The backlog

The backlog collects all tasks that have to be completed at a later date.
To move the task to the board you can click on the arrow-icon.
You can edit the task by clicking on it.

The Archive

In the archive, all tasks are stored that have already been completed.
It can be used to figure out who has completed which task.

The Trash

All tasks that are deleted end up in the recycle bin.
It prevents tasks from being deleted directly.
Deleting from the recycle bin cannot be undone.
