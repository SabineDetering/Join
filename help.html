Join is a Kanban Board. It visualises the project flow in agile project management.

This app was built as a training project at Developer Akademie. There is no security layer implemented. It is not supposed to be used with real life data.

Do not enter personal data.

Text about add task

The backlog is a collection of tasks and ideas that are not yet in the focus of implementation. Items can be moved either to the to-do-list or to the archive or be deleted.

The board holds all tasks in the current sprint. During the process of implementation and testing, tasks move from left to right through the stages of the board.

The archive can be used as a history of tasks that were not implemented or that are already done but not longer shown on the board.
